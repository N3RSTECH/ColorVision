# ColorVision

ColorVision adalah web sederhana untuk mendeteksi warna secara langsung menggunakan kamera perangkat. Semua proses dilakukan di browser tanpa mengirim data ke server.

## Fitur

- Scan warna secara real-time
- Menampilkan kode HEX, RGB, dan HSL
- Berjalan langsung di browser
- Ringan dan responsif

## Privasi

ColorVision meminta izin kamera hanya untuk mendeteksi warna yang terlihat pada kamera.

- Tidak mengambil atau menyimpan foto.
- Tidak merekam video.
- Tidak mengirim data kamera ke server.
- Tidak mengumpulkan data pribadi pengguna.

Semua proses dilakukan secara lokal di perangkat Anda.
